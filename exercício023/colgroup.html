utilizando o colgroup para destacar colunas

no html5 foi criada a tag COLGROUP que é uma maneira de destacar as colunas inteiras das tabelas

dentro da tag colgroup se insere a tag col, uma para cada coluna, e em cada uma se atribui uma classe
